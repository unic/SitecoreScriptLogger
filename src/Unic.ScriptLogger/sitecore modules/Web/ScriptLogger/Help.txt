This is the help for the Sitecore ScriptLogger module. Currently the following commands/urls are available:

/scriptlogger                    shows the newest default log file, this is configured within the config file
/scriptlogger/help               shows this help page
/scriptlogger/listfiles          list all available files
/scriptlogger/file/<filename>    get the content of a specific file

NOTE
----
Please make sure that you are logged in as an administrator. The login is currently not part of this module.
